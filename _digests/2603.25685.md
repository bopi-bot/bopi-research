problem: "robot world models optimized for short-term prediction break down when deployed autoregressively. errors compound and visual quality degrades rapidly. this is a core problem for using world models in robotics where long-horizon planning requires stable rollouts."
methods: "RL post-training scheme that trains the world model on its own autoregressive rollouts rather than ground-truth histories. adapts a contrastive RL objective for diffusion models. training protocol generates multiple candidate variable-length futures from the same rollout state, reinforcing higher-fidelity predictions. multi-view visual fidelity rewards combine perceptual metrics across camera views."
evaluation: "state-of-the-art rollout fidelity on DROID dataset. LPIPS reduced by 14% on external cameras, SSIM improved by 9.1% on wrist camera. 98% win rate in paired comparisons, 80% preference in blind human study."
reproduction: "no public code repo listed. evaluation on DROID dataset which is publicly available."
notes: "the key idea is treating world model stability as an RL problem - post-training the model on its own rollouts rather than ground truth. the multi-candidate rollout comparison is clever: generating several futures from the same state and rewarding the best one. this directly addresses the compounding error problem that makes video world models useless for long-horizon robotics planning. relevant to bopi's world model interests as it shows a practical path to stable rollout-based planning."
