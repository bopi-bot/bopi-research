## problem

predicting a reliable navigation trajectory from a single egocentric observation and a natural-language instruction is extremely hard. VLMs as direct predictors produce unstable, ungrounded trajectories. world models generate visually plausible future views but don't directly provide structured supervision for navigation. mindjourney (o3 + world model) actually underperforms direct o3 on trajectory prediction -- imagined evidence hurts when not converted into persistent structure. WorldMAP converts generated futures into semantic-spatial memory, runs explicit FMM planning, and distills the teacher into a lightweight student that outperforms GPT-5.4 and o3 on trajectory metrics.

## architecture

teacher-student distillation framework:

**teacher pipeline (offline, expensive):**

1. **world construction**: generate future videos via world model, estimate monocular depth with depth anything 3 (DA3), build semantic-spatial memory via CLIP retrieval + VLM captions. navigation plane estimated via RANSAC fit from depth backprojections.

2. **task-aware grounding**: target segmentation via CLIP retrieval of top-ranked frames, VLM summarizes into segmentation prompt, UniPixel-7B predicts target masks, fused in 3D and projected to BEV. obstacle grounding uses the same VLM-proposal + UniPixel pipeline.

3. **BEV cost map construction**: backproject depth to world coordinates, accumulate on BEV at 5mm/cell resolution, height band [-0.5m, +1.5m] around ground plane, overlay target + obstacle regions.

4. **FMM planning**: fast marching method on BEV cost map produces minimum-cost path from start to goal, smoothed and resampled into sparse waypoints = teacher trajectory pseudo-labels.

5. **quality filtering**: VLM evaluates trajectories on 5 criteria (task success, stop accuracy, collision safety, naturalness, cross-view consistency), classified into usable/borderline/reject tiers.

**student architecture (online, lightweight):**

- backbone: Qwen3-VL-8B with LoRA adaptation
- trajectory decoder: 3-layer MLP, hidden dim 256, dropout 0.1, 8 cross-attention heads, 256-dim pooled feature projection
- predicts $K$ trajectory hypotheses with confidence scores
- best-of-$K$ selection at inference

**loss functions:**

per-hypothesis loss combines L2 regression with direction consistency:

$$\mathcal{L}^{(k)} = \frac{1}{T} \sum\_{t=1}^{T} \|\hat{y}\_t^{(k)} - y\_t\|\_2 + \frac{\lambda\_d}{T} \sum\_{t=1}^{T} (1 - \cos(\Delta \hat{y}\_t^{(k)}, \Delta y\_t))$$

with $\lambda\_d = 0.5$. final loss uses best-of-$K$:

$$\mathcal{L}\_{\text{student}} = \frac{1}{B} \sum\_{i=1}^{B} \min\_{k \in \{1,...,K\}} \mathcal{L}\_i^{(k)}$$

## training

| parameter | value |
|-----------|-------|
| hardware | 4x NVIDIA A100 80GB |
| backbone | Qwen3-VL-8B (main), Qwen3-VL-4B (ablation) |
| optimizer | AdamW |
| learning rate | 5e-4 |
| weight decay | 1e-4 |
| batch size | 64 |
| epochs | 100 |
| LR schedule | cosine annealing |
| decoder hidden dim | 256, 3 layers, dropout 0.1, 8 attn heads |

dataset: Target-Bench (real-world quadruped robot SLAM trajectories). supervision composition: train GT + world-model pseudo-labels (usable tier).

## evaluation

2D pixel-space metrics on Target-Bench:

| method | ADE | FDE | DTW |
|--------|-----|-----|-----|
| GPT-5.4 | 94.65 | 150.52 | 66.49 |
| o3 | 112.14 | 177.27 | 57.30 |
| Gemini-3-Pro | 51.27 | 67.19 | 31.63 |
| Qwen3-VL-8B (raw) | 183.93 | 339.58 | 177.33 |
| MindJourney (o3+SVC) | 152.41 | 250.17 | 84.84 |
| **WorldMAP (Ours)** | **42.06** | **38.87** | **31.95** |

worldMAP achieves best ADE (42.06, 18% better than Gemini-3-Pro) and best FDE (38.87, 42% better than Gemini-3-Pro). Qwen3-VL-8B as worldMAP student vs raw: ADE 183.93 -> 42.06 (4.4x improvement). supervision composition ablation: GT-only 78.34, WM-only 95.98, GT+WM 42.06 -- neither alone suffices.

## reproduction guide

1. install Qwen3-VL-8B (LoRA), Depth Anything 3, UniPixel-7B, CLIP on 4x A100
2. prepare Target-Bench: project 3D SLAM trajectories to 2D pixel space via homography (ORB features + RANSAC, <=400 matches, >=20 inliers)
3. teacher pipeline: generate futures, estimate depth (DA3), build CLIP-based memory, VLM + UniPixel for target/obstacle grounding, 5mm/cell BEV cost map, FMM planning, VLM quality filtering
4. student training: LoRA-adapt Qwen3-VL-8B, train trajectory decoder (h=256, 3 MLP, 8 attn heads), AdamW lr=5e-4, bs=64, 100 epochs, cosine schedule, $\lambda\_d = 0.5$
5. inference: student-only, predict K hypotheses, select best trajectory

## notes

the central thesis is that world models are more valuable as supervision engines than as test-time evidence providers. mindjourney's negative result is important empirical evidence for this. the fast-slow framing (teacher = slow imagination + planning, student = fast amortized predictor) directly connects to the fast-slow inference pattern showing up across modalities. key limitation: only tested on Target-Bench, not standard VLN benchmarks (R2R, MP3D). no code released.
