## problem

end-to-end learned world models for embodied agents lack interpretability, provide no guarantees on physical constraint satisfaction, and struggle with multi-entity dynamic environments. the ERA framework proposes an alternative: represent the environment as structured semantic events, encode them with permutation-invariant networks into a latent space with lyapunov-stable dynamics, and perform decision-making via memory-augmented retrieval over a knowledge bank of prior experiences. the approach achieves 100% collision-free navigation on a uav collision avoidance task using only 27k expert samples.

## architecture

the framework has four stages:

**event list.** the environment at time $t$ is a set of $N\_t$ events $E\_t = \{e\_t^i\}\_{i=1}^{N\_t}$, where each $e\_t^i \in \mathbb{R}^k$ encodes object-id, position, kinematic state, environment constraints, ego state, and target context. updated at $\Delta t = 20$--$50$ ms.

**event encoder.** a permutation-invariant encoder (deepsets or transformer) compresses the event list into a latent code $z\_t = f(E\_t)$, $z\_t \in \mathbb{R}^d$.

**event dynamics.** latent transition with contractive stability:

$$z\_{t+1} = \Psi \cdot z\_t + \Gamma \cdot a\_t + \varepsilon\_t, \quad \rho(\Psi) < 1$$

lyapunov constraint: $\|z\_{t+1}\|^2 - \|z\_t\|^2 < 0$, enforced by singular value clamping on $\Psi$ after each optimization step.

**retrieval-augmented action.** approximate nearest neighbor search via FAISS IVF-Flat:

$$N\_k(z\_t) = \{(z\_i, a\_i) \mid z\_i \in \text{Top-}k(\text{sim}(z\_t, z\_i))\}$$

actions computed via softmax attention weighting:

$$a\_t = \sum\_{i \in N\_k} w\_i \cdot a\_i, \quad w\_i = \frac{\exp(\text{sim}(z\_t, z\_i) / \tau)}{\sum\_j \exp(\text{sim}(z\_t, z\_j) / \tau)}$$

**clustered bayesian selection** prevents the "average-to-collision" failure mode: retrieved maneuvers are clustered by directional cosine similarity, the highest-weight cluster wins via bayesian estimation, and averaging happens only within the winning cluster.

total complexity: $O(d \cdot \log \|M\|)$, dominated by retrieval.

## training

two-stage training:

**pre-training loss:**

$$\mathcal{L}(\theta\_1) = \lambda\_m \cdot \mathcal{L}\_{\text{metric}} + \lambda\_i \cdot \mathcal{L}\_{\text{imitation}}$$

where $\mathcal{L}\_{\text{metric}}$ ensures latent distances approximate physical distances, and $\mathcal{L}\_{\text{imitation}}$ is supervised trajectory following (dagger-style).

**online training loss:**

$$\mathcal{L}(\theta\_2) = \lambda\_p \cdot R\_{\text{phys}}(w, z) - \lambda\_r \cdot J\_{\text{perf}}$$

physics-consistency regularizer $R\_{\text{phys}}$ penalizes physics-inconsistent retrievals. performance objective optimized via policy gradient.

hardware: NVIDIA RTX 4090 (workstation), Jetson Orin Nano (edge). 27,075 expert samples from virtual potential field supervisor. 5 adversarial curriculum episodes with sigmoid-based difficulty scheduling. sub-millisecond inference latency within 20-50ms control loop.

## evaluation

on uav collision avoidance in nvidia isaac sim:

| metric | value |
|--------|-------|
| success rate | 100% across all 5 curriculum episodes |
| collisions | 0 |
| avg trajectory length | ~680 steps per 100m |
| final converged loss | ~2.6 |
| peak loss (adversarial) | ~77.9 (episode 2) |

key finding: decision robustness is decoupled from loss minimization -- the system converges to behaviorally optimal regime despite non-stationary adversarial loss landscapes. without physics-consistency regularizer, retrieved maneuvers exhibited high-frequency oscillations exceeding actuator limits.

## reproduction guide

1. install nvidia isaac sim, create 100m navigation corridor with dynamic intruder spawning, set $\Delta t = 0.05$s
2. implement VPF supervisor for collision-free trajectory generation, collect 27k demonstrations
3. implement event encoder as deepsets or transformer, pre-train with imitation + metric loss
4. build FAISS IVF-Flat index on latent codes
5. implement adversarial curriculum (sigmoid + adaptive success-rate term), train 5 episodes
6. project $\Psi$ onto contractive manifold via SVD clamping each step
7. filter actions by lyapunov constraint $V(z\_{t+1}) < V(z\_t)$

missing details: latent dimension $d$, event feature dimension $k$, number of clusters for bayesian selection, temperature $\tau$, optimizer/lr/batch size, $\lambda$ hyperparameters, wall-clock training time, perceiver architecture.

## notes

the retrieval-augmented paradigm is notably data-efficient: 27k expert samples vs millions for end-to-end approaches. the lyapunov stability constraint is a rare example of control-theoretic guarantees in learning-based planning. the clustered bayesian selection for resolving multi-modal action distributions is elegant -- this is the same "average to collision" problem that comes up in diffusion policy when averaging divergent trajectory samples. the framework effectively adapts RAG from NLP to physical decision-making, replacing text chunks with physics-grounded maneuver experiences. scalability beyond $10^5$ knowledge bank entries on edge hardware remains an open problem (proposes HNSW indexing).
