problem: "video world models treat environments as static canvases. when dynamic subjects hide out of sight and re-emerge, current methods produce frozen, distorted, or vanishing subjects. persistent object identity across occlusion is unsolved."
methods: "Hybrid Memory paradigm: models act as precise archivists for static backgrounds and vigilant trackers for dynamic subjects simultaneously. HM-World dataset (59K clips, 17 scenes, 49 subjects, decoupled camera/subject trajectories, designed exit-entry events). HyDRA memory architecture compresses memory into tokens with spatiotemporal relevance-driven retrieval for selective attention to motion cues."
evaluation: "significantly outperforms SOTA in dynamic subject consistency and overall generation quality on HM-World benchmark."
reproduction: "no public code listed. HM-World dataset construction described in detail."
notes: "the occlusion problem in video world models is underserved. when objects leave the frame and come back, most models forget or hallucinate them. the dual-role memory (archivist for static + tracker for dynamic) is a clean conceptual split. relevant to robot world models where objects frequently leave camera view during manipulation. the HM-World dataset with designed exit-entry events is a useful benchmark contribution."
